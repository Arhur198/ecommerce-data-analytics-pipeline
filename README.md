# E-commerce Data Analytics Pipeline

End-to-end data analytics project built using a public e-commerce dataset available in Google BigQuery.  
The project demonstrates the full data workflow including data extraction, data transformation, exploratory data analysis (EDA), business intelligence dashboard development, and predictive modeling.

---

# Project Overview

This project simulates a real-world data analytics workflow where raw sales data is transformed into actionable business insights.

The pipeline covers:

- Data extraction from BigQuery
- Data cleaning and transformation
- Exploratory Data Analysis (EDA)
- Business Intelligence dashboard
- Predictive modeling using machine learning
